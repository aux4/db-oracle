# aux4-db-oracle

Oracle database tools using oracledb client for the aux4 ecosystem.

This package provides an Oracle database driver that follows the aux4 db driver specification outlined in [db/package/README.md](../db/package/README.md).

- [aux4 hub](https://hub.aux4.io/r/public/packages/aux4/db-oracle)
- [package documentation](./package/README.md)

## Installation

```bash
aux4 aux4 pkger install aux4/db-oracle
```

## Development

```bash
# Install dependencies
npm install

# Build the executable
npm run build

# Run in development mode
npm run dev
```
